# JD-1: 矩阵链乘法（Removing Boxes Problem）

## 题目描述

给定一个数组 B，其中包含 n 个整数。题目要求计算将数组中的所有相邻元素合并成一个元素所需的最小操作次数。

**操作规则**：
- 每次操作可以选择数组中相同的相邻元素进行合并
- 合并后数组长度减1
- 直到数组只剩一个元素为止

**示例**：
```
输入: n=3, B=[1,2,1]
输出: 1
解释: 合并 B[0] 和 B[2]（尽管不相邻，通过其他操作可以使其相邻）
```

## 解题思路

### 核心思想
这是一个经典的动态规划问题。使用 `dp[i][j]` 表示将子数组 `B[i:j+1]` 合并到一个元素所需的最少操作次数。

### 状态转移
1. **基础情况**：`dp[i][i] = 1`（单个元素本身算1个元素）

2. **区间长度递推**：
   - 首先尝试不合并边界元素：`dp[i][j] = 1 + dp[i+1][j]`
   - 如果 `B[i] == B[j]`，可以通过合并中间元素使得 `B[i]` 和 `B[j]` 相邻并合并：
     `dp[i][j] = min(dp[i+1][j], dp[i][j-1])`
   - 否则，尝试在中间分割点进行合并：
     `dp[i][j] = min(dp[i][k] + dp[k+1][j])` for all `k in [i, j-1]`

### 时间复杂度
- **时间**: $O(n^3)$ - 三层嵌套循环
- **空间**: $O(n^2)$ - 二维DP数组

## 代码实现

```go
// 动态规划解决矩阵链乘法问题
// dp[i][j] 表示合并 B[i:j+1] 所需的最少操作次数
// 最终答案: dp[0][n-1]
```

## 关键要点

1. **相同元素优化**：如果头尾相同，可以通过合并中间元素使其相邻
2. **最优子结构**：问题具有最优子结构性质，适合用动态规划求解
3. **区间DP的标准套路**：按照区间长度从小到大进行递推

- 理解区间动态规划的应用
- 掌握二维DP状态设计
- 学习如何利用特殊性质（如相同元素）进行优化
