# 360/2: 删除一个元素后的最长递增子序列

## 题目描述

给定一个整数数组 h，需要找到删除数组中恰好一个元素后，最长递增子序列（LIS）的长度。

**问题定义**：
- 可以选择删除数组中的任意一个元素（或不删除）
- 删除后找到最长严格递增子序列
- 返回最长的子序列长度

**示例**：
```
输入: h = [1, 3, 4, 2, 5]
输出: 4
解释: 
- 如果删除下标1的元素3，得到 [1, 4, 2, 5]，LIS为 [1, 2, 5]或[1, 4, 5]，长度为3
- 如果删除下标2的元素4，得到 [1, 3, 2, 5]，LIS为 [1, 3, 5]或[1, 2, 5]，长度为3
- 如果删除下标3的元素2，得到 [1, 3, 4, 5]，LIS为 [1, 3, 4, 5]，长度为4
```

## 解题思路

### 核心思想
使用两个动态规划数组：
- `dp[i]`: 以位置i结尾的最长递增子序列长度
- `dpReverse[i]`: 从位置i开始的最长递增子序列长度

通过这两个数组，可以快速计算删除任意位置后的LIS长度。

### 算法步骤

1. **正向DP（从左到右）**：
   ```
   dp[i] = max(dp[j] + 1) for all j < i where h[j] < h[i]
   dp[0] = 1
   ```
   表示以h[i]结尾的最长递增子序列长度

2. **反向DP（从右到左）**：
   ```
   dpReverse[i] = max(dpReverse[j] + 1) for all j > i where h[i] < h[j]
   dpReverse[n-1] = 1
   ```
   表示从h[i]开始的最长递增子序列长度

3. **枚举删除位置**：
   对于每个位置i，计算删除该位置后的最长LIS：
   
   - **完全删除第i个元素**（跨越i）：
     如果 `h[i-1] < h[i+1]`，可以连接：
     `dp[i-1] + dpReverse[i+1] + 1`
   
   - **只保留左边LIS**（删除i及右边）：
     `dp[i-1] + 1`
   
   - **只保留右边LIS**（删除i及左边）：
     `dpReverse[i+1] + 1`

4. **返回最大值**

### 时间复杂度
- **时间**: $O(n^2)$ - DP计算为O(n²)，遍历删除位置为O(n)
- **空间**: $O(n)$ - 两个DP数组

### 优化可能性
- 使用二分搜索 + 贪心可以将DP优化到 $O(n \log n)$
- 但该解法更直观易懂

## 代码实现

```go
func maxLong(arr []int) int {
    n := len(arr)
    if n <= 1 {
        return n
    }
    
    // 正向DP: 以i结尾的LIS长度
    dp := make([]int, n)
    for i := range dp {
        dp[i] = 1
    }
    for i := 1; i < n; i++ {
        for j := 0; j < i; j++ {
            if arr[j] < arr[i] {
                dp[i] = max(dp[i], dp[j]+1)
            }
        }
    }
    
    // 反向DP: 从i开始的LIS长度
    dpReverse := make([]int, n)
    for i := range dpReverse {
        dpReverse[i] = 1
    }
    for i := n - 2; i >= 0; i-- {
        for j := i + 1; j < n; j++ {
            if arr[i] < arr[j] {
                dpReverse[i] = max(dpReverse[i], dpReverse[j]+1)
            }
        }
    }
    
    // 枚举删除位置
    maxlen := 0
    for i := 0; i < n; i++ {
        // 情况1: 跨越i连接（如果可能）
        if i > 0 && i < n-1 {
            if arr[i-1] < arr[i+1] {
                maxlen = max(maxlen, dp[i-1]+dpReverse[i+1]+1)
            }
        }
        // 情况2: 只保留左边
        if i > 0 {
            maxlen = max(maxlen, dp[i-1]+1)
        }
        // 情况3: 只保留右边
        if i < n-1 {
            maxlen = max(maxlen, dpReverse[i+1]+1)
        }
    }
    
    return maxlen
}
```

## 重要细节

### 为什么要跨越i？
删除位置i后，如果 `arr[i-1] < arr[i+1]`，可以直接连接左边的LIS和右边的LIS，形成新的递增子序列。

### 三种情况的含义
- **跨越**：通过删除i来桥接左右两部分
- **保留左边**：删除i及其右边，保留左边的LIS
- **保留右边**：删除i及其左边，保留右边的LIS

这三种情况涵盖了所有可能的删除策略。

## 复杂度对比

**朴素算法**：
- 对每个位置进行删除，重新计算LIS
- 时间：$O(n \times n^2)$ = $O(n^3)$

**优化算法**：
- 预计算正向和反向DP
- 时间：$O(n^2)$
