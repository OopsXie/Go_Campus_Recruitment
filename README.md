## 题目目录说明

### 京东 (JD)
- **[JD-1](./JD-1/)**: 矩阵链乘法动态规划问题
- **[JD-2](./JD-2/)**: 网格最短路径问题（带方向转换）

### 小米 (Xiaomi)
- **[xiaomi-1](./xiaomi-1/)**: 一元一次方程求解
- **[xiaomi-2](./xiaomi-2/)**: 二叉树平衡性检查
- **[xiaomi-2026-1](./xiaomi-2026-1/)**: 数组滑动窗口最大和
- **[xiaomi-2026-2](./xiaomi-2026-2/)**: 物品选择最优值问题

### 360
- **[360/1](./360/1/)**: 字符串变换可行性判断
- **[360/2](./360/2/)**: 删除一个元素后的最长递增子序列

### 最右
- **[最右/1](./最右/1/)**: 滑动窗口求最大连续人数
- **[最右/2](./最右/2/)**: 网格迷宫最短路径问题

## 外部资源

- **Go-Interview**: 来自网络的Go语言面试资料集合
- **golang-guide**: 综合的Go语言学习指南和资源

## 使用说明

每个题目目录下都包含：
- `main.go`: 题目的完整解决方案
- `README.md`: 题目描述和解题思路说明

### 运行示例

```bash
cd JD-1
go run main.go
```

## 学习建议

1. 逐个浏览题目，理解题目要求
2. 阅读相应的README文档，学习解题思路
3. 尝试自己实现解决方案
4. 参考给出的代码实现，对比不同的解题方法
5. 分析时间复杂度和空间复杂度

## 主要算法类型

- **动态规划 (DP)**: JD-1, xiaomi-2026-1, xiaomi-2026-2, 最右/2
- **图论/BFS**: JD-2
- **字符串处理**: xiaomi-1, 360/1
- **树论**: xiaomi-2
- **递增子序列**: 360/2, 最右/1
- **滑动窗口**: xiaomi-2026-1, 最右/1
