# Xiaomi-2026-1: 数组滑动窗口最大和

## 题目描述

给定一个包含 n 个非负整数的数组 a，和一个窗口大小 l。需要找出所有长度为 l 的连续子数组的最大和。

**问题定义**：
- 数组长度为 n，窗口大小为 l
- 需要计算所有可能的长度为 l 的连续子数组和
- 返回这些和中的最大值

**示例**：
```
输入: n=5, l=3, a=[1, 2, 3, 4, 5]
输出: 12
解释: 最大的长度为3的连续子数组是 [3, 4, 5]，和为12
```

## 解题思路

### 核心思想
使用滑动窗口（Sliding Window）技术，在 O(n) 时间内计算所有窗口的最大和。

### 算法步骤

1. **计算首个窗口的和**：
   ```
   windowSum = a[0] + a[1] + ... + a[l-1]
   maxSum = windowSum
   ```

2. **滑动窗口**：
   - 对于每个新位置 i (从 l 到 n-1)：
   - 移除左边界元素：`windowSum -= a[i-l]`
   - 添加新的右边界元素：`windowSum += a[i]`
   - 更新最大值：`maxSum = max(maxSum, windowSum)`

3. **返回最大值**

### 时间复杂度
- **时间**: $O(n)$ - 单次遍历数组
- **空间**: $O(1)$ - 仅使用常数级额外空间

### 关键优化点

与朴素做法相比：
- **朴素算法**: $O(n \times l)$ - 对每个窗口重新计算和
- **优化算法**: $O(n)$ - 通过只移除/添加一个元素

## 代码实现

```go
func maxConsecutiveSum(nums []int, k int) int {
    if len(nums) < k || k <= 0 {
        return 0
    }
    
    // 计算初始窗口和
    windowSum := 0
    for i := 0; i < k; i++ {
        windowSum += nums[i]
    }
    
    maxSum := windowSum
    
    // 滑动窗口
    for i := k; i < len(nums); i++ {
        windowSum = windowSum - nums[i-k] + nums[i]
        if windowSum > maxSum {
            maxSum = windowSum
        }
    }
    
    return maxSum
}
```

## 实际应用示例

**场景**：AI摄像头监控人流

```
人数记录: [2, 1, 5, 1, 2, 3, 4, 1, 2, 1]
窗口大小: 3分钟

处理过程:
- [2, 1, 5] → sum = 8
- [1, 5, 1] → sum = 7
- [5, 1, 2] → sum = 8
- [1, 2, 3] → sum = 6
- [2, 3, 4] → sum = 9 (最大)
- [3, 4, 1] → sum = 8
- [4, 1, 2] → sum = 7
- [1, 2, 1] → sum = 4

答案: 9
```

## 关键要点

1. **窗口大小固定**：关键特性，用于固定空间计算
2. **增量更新**：每次只更新一个元素
3. **单次遍历**：整个过程只遍历数组一次
4. **应用场景**：监控、时间序列分析、网络流量统计等
